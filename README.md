# InsightPipe

## 📖 项目简介 (Introduction)

**InsightPipe** 是一个实验性的个人知识管理工具，旨在通过记录学习过程中遇到的疑问及其 AI 解答，构建一个且不仅限于“问答对”的长期知识库。

核心理念是：**将疑惑转化为资产。**

很多时候，我们在学习中产生的疑惑和 AI 给出的精彩解答，往往在对话窗口关闭后就被遗忘。InsightPipe 试图捕捉这些流失的价值，通过标准化的 Prompt 和归档流程，将它们保存为 Markdown 文档。这不仅是对当下的解惑，更是对未来的一笔宝贵知识投资。

## 🎯 核心动机 (Motivation)

1.  **知识沉淀**：记录学习路径上的障碍与解决方案，形成个人专属的 FAQ 知识库。
2.  **Prompt 复用**：通过预定义的优质 Prompt 模版，减少重复输入，提高向 AI 提问的效率和质量。
3.  **零成本/低依赖**：MVP 阶段不依赖昂贵的 API 调用或本地算力，仅需作为“中间件”衔接思维与 AI 对话框。
4.  **长期主义**：一年或多年后回看，这些记录将成为个人成长的见证，利用 RAG 等技术可挖掘更多价值。

## ⚙️ 工作流 (Workflow - MVP)

目前的 MVP (Minimum Viable Product) 版本采用“人机协同”的半自动化模式：

1.  **输入疑惑**：而在 InsightPipe 中输入你不理解的概念或问题。
2.  **生成提示词**：系统自动套用预设的高效 Prompt 模版，生成一段标准化的提问文本。
3.  **AI 交互 (Copy-Paste)**：
    *   用户复制 Prompt 到 ChatGPT / Gemini / Claude 等 AI 工具。
    *   AI 输出结构化的 Markdown 解答。
4.  **归档保存**：
    *   用户将 AI 的回答复制回 InsightPipe。
    *   系统自动将其保存为标准 Markdown 文件（支持自动命名、分类）。

## 🔮 未来规划 (Future)

*   **本地知识库检索 (RAG)**：基于积累的 Markdown 文档，构建本地向量库，实现“与过去的自己对话”。
*   **知识图谱**：分析文档间的关联，可视化展示知识盲区与连接点。
*   **自动化集成**：接入 API，实现一键问答与归档（视成本与需求而定）。
*   **创意开发**：利用积累的高质量数据进行微调或个性化助手训练。
